GraphToys:

A simple graph algorithms program to practice some basic systems programming
challenges.

The challenges include:
* JSON parser and serializer
* Graph parser from JSON parser output
* Graph algorithms
* Custom memory allocators tailored for usage pattern
* CLI and TCP server executable
* Test suite for testing each component
* Modular build system with multiple components

Each component will have headers acting as its spec, tests
will be written by only referencing said specs.
